🎮 GameDev AI Assistant

Your Personal AI Mentor for Game Development
Built with Llama 3.2 (Ollama) + Streamlit

A fully local AI chatbot designed to assist game developers with Unity, Unreal Engine, gameplay systems, optimization, and design decisions — without relying on external APIs.

✨ Features
🧠 Runs Locally — No internet required after setup
💬 Conversation Memory — Maintains chat context
🎯 Game Dev Focused — Tailored for real-world dev problems
⚡ Fast & Lightweight UI — Built using Streamlit
💸 Completely Free — No API keys or subscriptions

📦 Tech Stack
LLM: Llama 3.2 via Ollama
Frontend: Streamlit
Backend: Python
Runtime: Local (offline-first)

🚀 Installation & Setup
1️⃣ Install Ollama

Download and install Ollama from:
👉 https://ollama.com/download

Then pull the model: ollama pull llama3.2

2️⃣ Clone the Repository : 
git clone https://github.com/aaniketdarve-21/gamedev-ai-assistant.git
cd gamedev-ai-assistant

3️⃣ Install Dependencies:
pip install streamlit ollama

4️⃣ Run the Application:
streamlit run chatbot.py

Your browser will open automatically 🚀
